Hackintosh

A Hackintosh is a colloquial term used to describe a non-Apple computer (usually a PC) that has been modified or configured to run macOS, Apple's operating system. In essence, it's a system built with off-the-shelf PC hardware components but set up in a way that allows it to run macOS, which is typically designed to run only on Apple's own hardware.
